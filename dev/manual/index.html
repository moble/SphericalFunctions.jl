Calculators

Typically, when calculating special functions, we will use recurrence relations along with some coefficients â€” which frequently requires significant setup processing. That processing can be cached, so that the calculations themselves consist primarily of memory accesses and simple arithmetic. For this reason, we use "calculator" objects, which will be constructed with some indication of the largest indices you expect to use. The calculators can then be called repeatedly for specific values of the arguments, which will compute the function values for all (or some subset of) indices.
